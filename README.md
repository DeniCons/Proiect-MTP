# Proiect-MTP – Gestionare Mașini

Acest proiect reprezintă o aplicație desktop realizată în **C#**, dezvoltată în cadrul disciplinei **Medii și Tehnologii de Programare (MTP)**.  
Aplicația are ca scop **gestionarea unui parc auto**, permițând administrarea informațiilor despre mașini într-un mod structurat și ușor de utilizat.

---

##  Funcționalități principale

Aplicația oferă următoarele funcționalități:

-  Adăugarea de mașini
-  Editarea informațiilor unei mașini existente
-  Ștergerea mașinilor
-  Afișarea listei de mașini
-  Salvarea și încărcarea datelor
-  Interfață grafică intuitivă

---

## Tehnologii utilizate
- Limbaj de programare: **C#**
- Platformă: **.NET**
- Tip aplicație: **Desktop (Windows Forms / WPF)**  
- Mediu de dezvoltare: **Visual Studio**
